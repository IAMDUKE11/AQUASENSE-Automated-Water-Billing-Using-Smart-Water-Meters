AquaSense
Project Overview
AquaSense is an IoT-based Automated water billing using smart water meters designed to monitor water consumption in real time, calculate water bills automatically, detect leaks, and provide alerts to users.

The system combines embedded hardware, a RESTful backend API, a Firebird database, and a web-based frontend dashboard.

Objectives
Monitor water flow in real time
Calculate water usage automatically
Generate bills based on consumption
Detect leak conditions
Store readings in a database
Provide historical monitoring
